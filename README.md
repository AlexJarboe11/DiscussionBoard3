# Breast Cancer Classification with Random Forest
This project used the Breast Cancer dataset from scikit-learn

Source: 'sklearn.datasets.load_breast_cancer'
Features: 30 numeric features extracted from digitzed images
Target: 0 = malignant, 1 = benign

# Model

- Algo: Random Forest Classifier
- Accuracy: ~96%
- ROC-AUC: ~.99

# Key Insights

The most important features were the mean texture, mean radius and worst perimeter
Random Forest gave a high accuracy and interpretability
